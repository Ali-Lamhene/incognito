# Mécaniques de Jeu

## 1. Création et Jonction
- **L'Hôte** : Crée un salon, définit les paramètres (nombre de défis par personne, thématique, durée).
- **Le Salon** : Génère un code unique (ex: `A7B2`).
- **Les Joueurs** : Rejoignent avec le code et un pseudonyme.

## 2. Attribution des Défis
Une fois la partie lancée :
- Chaque joueur reçoit un (ou plusieurs) défi(s) secret(s).
- Les défis sont variés : phrases à dire, gestes à faire, comportements à adopter.
- *Exemple* : "Faire un compliment sur les chaussures de quelqu'un", "Se toucher l'oreille gauche 3 fois en parlant", "Dire le mot 'Anticonstitutionnellement'".

## 3. Déroulement de la Partie
- La partie se joue "en arrière-plan" de la vie réelle.
- Un joueur qui pense avoir accompli son défi le valide sur l'application.
- Un joueur qui pense avoir surpris quelqu'un en train d'accomplir son défi clique sur "Démasquer".

## 4. Système de Points
- **Défi réussi** : +10 points.
- **Démasquer quelqu'un avec succès** : +15 points.
- **Se faire démasquer** : 0 point pour le défi (ou malus de -5).
- **Fausse accusation** : -5 points pour l'accusateur.

## 5. Fin de Partie
- La partie se termine quand :
    - Le temps est écoulé.
    - Tous les défis ont été tentés.
    - L'hôte décide d'arrêter.
- Un écran de résultats affiche le classement final et l'historique des défis réussis/ratés.
